0. Install Node and NPM

1. Install Angular CLI
npm install -g @angular/cli

2. Download/clone https://github.com/glarimy/glarimy-angular

3. Move to the project glarimy.ng.01
4. Install the libraries
npm install
5. Build and run the application
npm start
6. Check the results on the browser
http://localhost:4200
